# Hooks

# useState

const [value, setValue] = useState(defaultValue);
  • value는 상태의 값
  • setValue는 상태를 설정하는 함수
  • useState함수의 파라미터는 상태 기본값

# useEffect

리액트 컴포넌트가 렌더링될 때마다 특정 작업을 수행하도록 설정할 수 있는 Hooks
componentDidMount와 componentDidUpdate를 합친 형태

useEffect(() => {
  console.log("마운트될 때만 실행");
}, []);

함수의 두번째 파라미터로 비어있는 배열을 넣어주면 맨 처음 렌더링 될 때만 실행
componentDidMount와 비슷.

useEffect(() => {
  console.log("target이 업데이트 될 때 실행");
}, [target]);

두번째 파라미터의 배열에 체크하고자 하는 값을 넣으면 그 값이 변할 때 컴포넌트 렌더링
componentDidUpdate와 비슷.

useEffect(()=>{
    ...
    return ()=>{
        console.log('뒷정리를 하는 부분')
    }
})

return 이후의 함수를 통해서 업데이트 직전에 뭔가를 처리.
componentWillUnmount와 비슷

# useReducer

리듀서를 활용한 상태관리에 사용되는 Hooks
리듀서는 현재 상태, 업데이트를 위한 정보를 답은 액션값을 전달받아 새로운 상태를 반환하는 함수
불변성을 지켜줘야 함

const [state, dispatch] = useReducer(reducer, { value: 0 });

useReducer의 첫번째 파라미터는 리듀서함수, 두번째 파라미터는 리듀서의 기본값
이 hooks는 현재 상태를 가리키는 상태, state와 액션을 발생시키는 함수, dispatch(action)를 받아 옴

# useMemo

함수형 컴포넌트 내부에서 발생하는 연산을 최적화할 수 있음. 렌더링하는 과정에서 특정 값이 바뀌었을 때만 연산을 실행하고, 원하는 값이 바뀌지 않으면 이전 값을 그대로 사용하는 방식

const target = useMemo(() => setTarget(value), [value]);

value값이 바뀌었을 때만 target값을 업데이트한다.

# useCallback

useMemo와 비슷. 렌더링 성능 최적화해야 하는 상황에 사용. 이벤트 핸들러 함수를 필요할 때만 생성할 수 있음
첫번째 파라미터는 생성하고 싶은 함수, 두번째 파라미터는 배열, 이 배열에 감시할 값을 명시
숫자, 문자열, 객체처럼 일반 값을 재사용하려면 useMemo
함수를 재사용하려면 useCallback

useCallback(() => {
  console.log("hello world");
}, []);

useMemo(() => {
  const fn = () => {
    console.log("hello world");
  };
  return fn;
}, []);

# useRef

함수형 컴포넌트에서 ref를 쉽게 사용할 수 있도록 해주는 Hooks
useRef를 사용하여 ref를 설정하면 useRef를 통해 만든 객체 안의 current값이 실제 엘리먼트를 가리킴
렌더링과 관계없이 바뀔 수 있는 로컬 변수로도 사용 가능

import React, { useRef } from "react";

const RefSample = () => {
  const id = useRef(1);
  const setId = (n) => {
    id.current = n;
  };
  return <div>refSample</div>;
};
export default RefSample;

# useContext

Consumer 컴포넌트를 사용하지 않고도 부모 컴포넌트로부터 전달된 컨텍스트 데이터 사용 가능

const UserContext = React.createContext();
const user = { name : 'dfjdk', age : 23};

const ParentComponent = ()=>{
  return (
  	<UserContext.Provider value={user}>
    	<ChildComponent />
    </UserContext.Provider>
  )
}

const ChildComponent = ()=>{
  const user = useContext(UserContext);
  console.log(`user: ${user.name}`);
  
  return (
  	<div>
    	<UserContext.Consumer>
      	...
      </UserContext.Consumer>
    </div>
  )
}
