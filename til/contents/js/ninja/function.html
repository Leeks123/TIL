# 함수가 핵심이다

# 자바스크립트의 함수는 1종 객체이다.

  • 리터럴로 생성 가능
  • 변수, 프로퍼티 할당 가능
  • 매개변수로 전달 가능
  • 함수의 결과로 반환 가능
  • 프로퍼티와 메서드 가질 수 있음

객체와 함수의 차이라고 한다면 함수는 호출이 가능하다는 것

# 호출

# 매개변수, 인자

  • 매개변수보다 많은 인자가 제공되었다면, '남은' 인자들은 매개변수에 할당되지 않음
  • 인자보다 매개변수가 많은 경우, 인자와 부합하지 않은 매개변수는 undefined 로 설정됨
  • 모든 함수 호출에는 암묵적으로 arguments 와 this 매개변수가 넘어 옴
    • arguments 매개변수는 함수에 전달된 모든 인자들을 담은 컬렉션 배열은 아니지만 배열처럼 인덱스 접근가능. length 프로퍼티, for 루프 사용가능
    • this 는 암묵적으로 함수호출과 관계된 객체를 참조. 함수 컨텍스트

# 함수로 호출

  • ( ) 연산자를 사용하여 호출하는 방법. 가장 기본적인..
  • 암묵적으로 모든 함수의 소유자는 window 객체

# 메서드로 호출

  • 함수를 객체의 프로퍼티로 할당하고, 그 프로퍼티를 사용하여 '참조에 의한 호출'이 일어나면, 함수는 그 객체의 메서드로 호출
  • 이때 this 가 함수가 속한 객체가 된다.
function greet() {
  console.log('hello',this);
}
greet(); // hello Window {...}

const obj = {
  fn : greet
}
obj.fn(); // hello {fn: f}

# 생성자로 호출

  • new 키워드를 붙여서 함수를 호출하는 방법
  • 비어있는 객체가 새로 생성됨
  • 새로 생성된 객체는 this 매개변수로 생성자 함수에 전달되고, 따라서 생성자의 함수 컨텍스트가 됨
  • 어떤 명시적인 반환값이 없다면, 새로 생성된 객체가 생성자의 값으로 반환됨
  • 생성자 함수라면 대문자로 시작하는 명명법으로...
