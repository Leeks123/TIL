# Array

자바스크립트의 배열은 일반적인 배열의 동작을 흉내 낸 특수 객체.

console.log(Object.getOwnPropertyDescriptors([1, 2, 3]));
/*
{
  '0': { value: 1, writable: true, enumerable: true, configurable: true },
  '1': { value: 2, writable: true, enumerable: true, configurable: true },
  '2': { value: 3, writable: true, enumerable: true, configurable: true },
  length: { value: 3, writable: true, enumerable: false, configurable: false }
}
*/

해시 테이블로 구현된 객체이므로 인덱스로 요소에 접근하는 경우 일반 배열보다 느리지만, 검색, 삽입, 삭제의 경우 빠른 성능

# length

length 프로퍼티는 기본적으로 배열에 요소를 추가하거나 삭제하면 자동적으로 갱신된다. 그러나 임의적으로 할당할 수도 있다.

const arr = [1, 2, 3];
console.log(arr.length); // 3

arr.length = 2;
console.log(arr); // [1,2]

arr.length = 5;
console.log(arr); // [ 1, 2, 3, <2 empty items> ]

자바스크립트는 희소배열을 문법적으로 허용한다. 그렇지만 희소배열을 생성하지 않도록 주의해야...

const sparse = [, 2, , 4];

console.log(sparse.length); // 4
console.log(sparse); // [ <1 empty item>, 2, <1 empty item>, 4 ]

# 배열 리터럴

const arr = [1, 2, 3];

# Array 생성자 함수

const arr = new Array(10);

// 1.
const err = new Array(4294967295); // RangeError.

// 2.
new Array(1, 2, 3); // [1,2,3] - 2.

// 3.
new Array({}); // [{}] - 3.
new Array([1, 2, 3]); // [[1,2,3]]

// 4.
Array(1, 2, 3); // [1,2,3] -4
  1. 생성자 함수의 인자는 0 ~ 2^32-1까지
  2. 2개 이상의 인자일 경우 인자를 요소로 갖는 배열을 생성
  3. 인자가 숫자가 아닐 경우 인자를 요소로 갖는 배열을 생성
  4. new가 없어도 동작한다.

# Array.of

ES6에서 도입됨
전달된 인자를 요소로 갖는 배열을 생성

Array.of(1); // [1]
Array.of(1, 2, 3); // [1,2,3]

# Array.from

ES6에서 도입됨

유사 배열 객체(length 프로퍼티를 가지는 객체) 또는 이터러블 객체를 인수로 전달받아 배열로 반환

Array.from({ length: 2, 0: "a", 1: "b" }); // [ 'a', 'b' ]
Array.from("Hello"); // [ 'H', 'e', 'l', 'l', 'o' ]

두번째 인자로 콜백함수를 전달해 요소를 채울 수도 있다.

Array.from({ length: 3 }, (_, i) => i); // 1,2,3

# 배열 메서드

# Array.isArray

Array 생성자 함수의 정적 메서드.

전달된 인자가 배열이면 true, 아니면 false

Array.isArray([]); // true
Array.isArray([1, 2]); // true

Array.isArray(); // false
Array.isArray({}); // false
Array.isArray(1); // false
Array.isArray(null); // false

# Array.prototype.indexOf

인자로 전달된 요소를 검색하여 인덱스를 반환

const arr = [1, 2, 3, 4];

arr.indexOf(1); // 0
arr.indexOf(0); // -1. 없으면 -1을 반환
arr.indexOf(4, 1); // 3. 인덱스1부터 검색을 시작한다

# Array.prototype.includes

ES7에 도입됨

배열 내에 특정 요소가 포함되어 있는지 확인하여 불린값 반환

const arr = [1, 2, 3, 4];

arr.includes(2); // true
arr.includes(0); // false
arr.includes(4, 1); // true. 두번째 인자는 검색시작 위치. 음수도 가능

indexOf는 존재 여부 확인시 반환값이 -1인지를 확인해야 하는점. 그리고 NaN 포함여부는 확인할 수 없다.

[NaN].indexOf(NaN) !== -1; // false
[NaN].includes(NaN); // true

# Array.prototype.push

const arr = [1, 2, 3];

console.log(arr.push(4, 5)); // 5
console.log(arr); // [1,2,3,4,5]

인자로 받는 값들을 배열의 마지막 요소에 추가하고 변경된 length 값을 반환한다.
원본배열을 직접 변경한다.

const arr = [1, 2];

arr[arr.length] = 3;
console.log(arr); // [1,2,3]

push 메서드는 성능이 좋지는 않다. length 프로퍼티를 활용해 직접 추가하는 것이 더 빠르다

const arr = [1, 2];

const newArr = [...arr, 3, 4];
console.log(newArr); // [1,2,3,4]

원본배열을 조작하지 않으려면 스프레드 연산자를 활용할 것..

# Array.prototype.pop

원본 배열에서 마지막 요소를 제거하고 제거한 요소를 반환한다.
빈배열일 경우 undefined를 반환한다.
원본 배열을 직접 변경한다.

const arr = [1, 2, 3];

console.log(arr.pop()); // 3
console.log(arr); // [1,2]

# Array.prototype.unshift

인수로 전달받은 모든 값을 원본 배열의 선두에 요소로 추가하고 변경된 length 프로퍼티 값을 반환
원본배열을 직접 변경한다.

const arr = [1, 2];

console.log(arr.unshift(0)); // 3
console.log(arr); // [0,1,2]

# Array.prototype.shift

원본 배열에서 첫번째 요소를 제거하고 제거한 요소를 반환.
원본 배열이 빈 배열이면 undefined를 반환
원본 배열을 직접 변경

const arr = [1, 2, 3];

console.log(arr.shift()); // 1
console.log(arr); // [2,3]

# Array.prototype.concat

인자로 전달된 값들을 원본 배열의 마지막 요소로 추가한 새로운 배열을 반환
원본 배열은 변경되지 않음

const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

let result = arr1.concat(arr2);

console.log(result); // [1,2,3,4,5,6]

# Array.prototype.splice

원본 배열의 중간에 요소를 추가하거나 중간에 있는 요소를 제거할 때 사용

const arr = [1, 2, 3, 4];

const result = arr.splice(1, 2, 20, 30);
// 인덱스1부터 2개를 제거하고 그자리에 20,30을 삽입한다

console.log(result); // [2,3]. 제거한 요소를 배열로 반환
console.log(arr); // [1,20,30,4]. 원본 배열을 조작

# Array.prototype.slice

인자로 전달된 범위의 요소들을 복사하여 배열로 반환한다.
원본 배열은 변경되지 않는다.

const arr = [1, 2, 3];

arr.slice(0, 1); // [1]. 인덱스0부터 인덱스1 이전까지 복사하여 반환

이때의 배열 복사는 얕은 복사

# Array.prototype.join

원본 배열의 모든 요소를 문자열로 변환한 후, 인자로 전달받은 문자열(구분자)로 연결한 문자열을 반환

const arr = [1, 2, 3, 4];

arr.join(); // '1,2,3,4';
arr.join(""); // '1234';
arr.join(":"); // '1:2:3:4'

# Array.prototype.reverse

원본 배열을 뒤집음.
원본 배열이 변경되고, 변경된 배열을 반환한다.

const arr = [1, 2, 3, 4];
const result = arr.reverse(); //

console.log(arr); // [4,3,2,1]
console.log(result); // [4,3,2,1]

# Array.prototype.fill

ES6에 도입됨

인자로 전달받은 값으로 배열을 채움
원본배열이 변경됨

const arr = [1, 2, 3, 4];

arr.fill(0);
console.log(arr); // [0,0,0,0]

# Array.prototype.flat

ES10에 도입됨

인자로 전달한 깊이 만큼 재귀적으로 배열을 평탄화. 기본값은 1

[1, [2, [3, [4]]]].flat(); // [1,2,[3,[4]]]
[1, [2, [3, [4]]]].flat(2); // [1,2,3,[4]]
[1, [2, [3, [4]]]].flat(3); // [1,2,3,4]
[1, [2, [3, [4]]]].flat(Infinity); // [1,2,3,4]

Infinity 로 인자를 전달받을 경우 중첩배열 모두 평탄화

# 배열 고차 함수

# Array.prototype.sort

오름차순 정렬
정렬 순서는 유니코드 코드 포인트의 순서를 따름. 따라서 숫자를 정렬할 떄는 비교 함수를 인자로 넘겨야 함

[2, 10].sort(); // [10,2]

[2, 10].sort((a, b) => a - b);

# Array.prototype.forEach

forEach 메서드의 콜백 함수는 호출한 배열의 요소값, 인덱스, 배열 자체(this)를 인자로 받는다.

[1, 2, 3].forEach((item, index, arr) => {
  console.log(item, index, arr);
});
/*
1 0 [ 1, 2, 3 ]
2 1 [ 1, 2, 3 ]
3 2 [ 1, 2, 3 ]
*/

# Array.prototype.map

콜백함수의 반환값들로 구성된 새로운 배열을 반환
호출배열과 생성배열은 1:1 매핑한다.

# Array.prototype.filter

콜백함수의 반환값이 true인 요소로만 구성된 새로운 배열을 반환

const number = [1, 2, 3, 4, 5];

const odds = number.filter((item) => item % 2);
console.log(odds); // [1,3,5]

# Array.prototype.reduce

콜백 함수의 반환값을 다음 순회 시에 콜백 함수의 첫 번째 인수로 전달하면서 콜백함수를 호출하여 하나의 결과값을 만들어 반환
첫번째 인자는 콜백함수, 두번째 인자는 초기값
콜백함수는 초기값 또는 콜백 이전 반환값, 배열 요소, 인덱스, this 를 인자로 받는다.

const sum = [1, 2, 3, 4].reduce(
  (accumulator, item, index, arr) => accumulator + item,
  0
);
console.log(sum); // 10

# Array.prototype.some

콜백함수의 반환값이 단 한번이라도 참이면 true, 모두 거짓이면 false를 반환

[5, 10, 15].some((item) => item > 10); // true
[5, 10, 15].some((item) => item < 0); // false

# Array.prototype.every

콜백함수의 반환값이 단 한번이라도 거짓이면 false, 모두 참이면 true를 반환

[5, 10, 15].every((item) => item > 3); // true
[5, 10, 15].some((item) => item < 10); // false

# Array.prototype.find

콜백함수의 반환값이 true인 첫 번째 요소를 반환

[1, 2, 2, 3].find((item) => item == 2); // 2

# Array.prototype.findIndex

콜백함수의 반환값이 true인 첫 번째 요소의 인덱스를 반환

[1, 2, 2, 3].findIndex((item) => item == 2); // 1

# Array.prototype.flatMap

const arr = ["hello", "world"];

arr.map((x) => x.split("")).flat();
// ['h','e','l','l','o','w','o','r','l','d']
arr.flatmap((x) => x.split(""));
// ['h','e','l','l','o','w','o','r','l','d']
